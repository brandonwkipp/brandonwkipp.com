October 9th, 2022

Loving Gitea now that I have it set up on my own server. Slowly picking back up on my personal development. Things have gone a bit out of whack, but I'm trying to practice minimalism so I'm getting rid of a lot of code for different reasons. Getting rid of this stuff makes me feel lighter. Feels like I'm actually unlearning stuff. In line with this, I recently deleted my Facebook. I kept an archive and find myself looking through all those messages in JSON format. I'm glad I kept the archive. A lot of good memories.
